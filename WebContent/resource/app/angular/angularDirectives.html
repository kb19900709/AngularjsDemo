Angularjs Life Cycles

  1. HTML 載入 Angularjs (或是選擇性載入其餘第三方類庫)
  2. 載入相關 javaScript 程式碼
  3. Document Ready Event
  4. 搜尋 ng-app 確認 app 作用邊界以及附屬之所有實體
  5. 進入編譯 (compile)
  6. 搜尋HTML上所有的 Directives 並執行 Directives 定義物件所需的程式碼
  7. 若第六點有載入或是建立新的HTML，則遞迴進入相關HTML執行第六點
  8. 所有的 Directives 編譯結束，並生成 Link 函式 (如果有定義)
  9. Angularjs 組合 Scope(範疇) 以及 Link 函式
  10. 若HTML有引用 Scope 上的變數，則顯示於 HTML上
  11. Angularjs  針對所有 Directives  以及綁定的變數加入 監視器 (watchers)，保證 UI 與 Model 一致
  12. 產生有資料有互動功能的 view

行為修飾 : ng-show、ng-hide ... etc

重用元件 : 樣板、描繪邏輯、工作邏輯
